# App Icons

This directory should contain the following icon files for PWA support:

- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

These icons will be used when users add the website to their home screen on mobile devices.

You can generate these icons from a single high-resolution source image (at least 512x512 pixels) using an online tool or image editing software.
